# ColpittsOscillator
Analyse des nichtlinearen Colpitts-Oszillators und Entwurf einer Regelung mithilfe von MATLAB/ Simulink

Das Ziel dieses Projekt ist es, nichtlineare Systeme am Beispiel des Colpitts-Oszillators zu analysieren.
Es handelt sich um eine Partnerarbeit im Rahmen des Moduls "Regelungstheorie - Nichtlineare Regelung" (Masterstudiengang). Daher wird die gegebene Aufgabenstellung als Richtlinie verwendet:
  - Verstehen Sie die Modellbildung
  - Berechnen Sie das Zustandsraummodel
  - Implementieren Sie die Regelstrecke in MATLAB/Simulink
  - Berechnen bzw. bestimmen Sie Gleichgewichtslagen der Systeme und deren Stabilität!
  - Bestimmen Sie Parameterwerte und Anfangsbedingungen, prüfen Sie deren physikalisch-technische Sinnfälligkeit
  - Stellen Sie in Simulationsexperimenten fest, ob sich typische Phänomene nichtlinearer Systeme einstellen
  - Wählen Sie nichtlineare Regelungskonzepte und entwerfen Sie Regler für das nichtlineare Model
  - Implementieren Sie ihn und beurteilen Sie das entstehende dynamische Verhalten
  - Wählen und implementieren Sie ein für Sie passendes Projektmanagement
